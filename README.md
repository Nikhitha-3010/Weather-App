# Weather App 🌦️

This is a simple and user-friendly Weather Application that provides real-time weather information for any city.

## 🔍 Features
- Get current weather details
- Search weather by city name
- Displays temperature, humidity, and weather conditions
- Clean and responsive UI
- Real-time data using API

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Weather API (OpenWeatherMap)

## ⚙️ How It Works
1. Enter the city name
2. Click on search
3. The app fetches weather data using API
4. Displays temperature and weather conditions

## 🚀 How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter city name and get weather details
